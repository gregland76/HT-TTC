Aide du calculateur HT-TTC

Cette aide explique rapidement comment utiliser l'application pour convertir un montant hors taxes en toutes taxes comprises, ou l'inverse, avec un taux de TVA personnalisable.

Utilisation rapide

  1. Saisissez un montant dans le champ HT pour obtenir automatiquement le TTC.
  2. Ou saisissez directement le TTC pour recalculer le HT.
  3. Modifiez le taux dans l'onglet Parametres si besoin.

Formules

TTC = HT × (1 + TVA / 100)

HT = TTC / (1 + TVA / 100)

Saisie des nombres

Les champs numeriques n'acceptent que les nombres. Vous pouvez utiliser une virgule ou un point pour les decimales, par exemple 12,50 ou 12.50.

Points d'attention

Un taux de TVA de -100 est interdit, car il rend le calcul impossible. Si une valeur est incomplete, le calcul attend une saisie valide avant de mettre a jour le resultat.

Le taux peut etre modifie a tout moment. Les calculs se mettent a jour automatiquement des qu'une valeur valide est saisie.

Exemple 1

Calcul HT vers TTC

Montant HT saisi : 100

Taux de TVA : 20

Calcul applique : 100 × (1 + 20 / 100)

TTC = 120,00 €

Exemple 2

Calcul TTC vers HT

Montant TTC saisi : 240

Taux de TVA : 20

Calcul applique : 240 / (1 + 20 / 100)

HT = 200,00 €

Exemple 3

Avec un autre taux

Montant HT saisi : 89,90

Taux de TVA : 5,5

Calcul applique : 89,90 × (1 + 5,5 / 100)

TTC = 94,84 €
